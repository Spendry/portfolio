## Adding a certificate

1. Add a row to the table with the issuer's verification URL in the verify cell.
2. If you want the PDF here too, drop it in this folder and link it: `[PDF](./cert-name.pdf)`.
3. Keep the table sorted by date, newest first.

Verification links age better than uploaded files. Lead with the link.
